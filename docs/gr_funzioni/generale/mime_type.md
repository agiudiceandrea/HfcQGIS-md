# mime_type

Restituisce il tipo mime dei dati binari.

## Sintassi

* mime_type(_bytes_)

## Argomento

* _bytes_ I dati binari

## Esempi

* `mime_type('<html><body></body></html>') → text/html`
* `mime_type(from_base64('R0lGODlhAQABAAAAACH5BAEKAAEALAAAAAABAAEAAAIAOw==')) → image/gif`
