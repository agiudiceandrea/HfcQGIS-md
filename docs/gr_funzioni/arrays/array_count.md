# array_count

Conta il numero di occorrenze di un dato valore in un array.

## Sintassi

array_count(array,value)

## Argomenti

* _array_ un array
* _value_ un valore da contare

## Esempi

* `array_count(array('a', 'b', 'c', 'b'), 'b') â†’ 2`
